SEAEP 2 -- Course
=================

Abstract
--------
Define course model.

Course
------
- Course Catalog
 - Search/browse
 - Course info
 - Instructor profiles
- Class
 - syllabus/schedule
 - forum viewing/posting
 - assignment grade submissions
 - peer grading submission
 - lecture/subtitle info
 - student progress, activity
 - user roles/permissions
 - quiz question import/export
 - ... and more
- Student
 - account/profile
 - enrolled courses
 - send message
- Auth
 - OpenID (identity)
 - OAuth (access)
